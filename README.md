# PRAROZ Website Enhancement Project (Part 3)

## 📌 Project Overview
This project is part of a web development assignment focused on improving an existing website by adding advanced functionality, interactivity, and Search Engine Optimisation (SEO).

The final output is a single HTML page that demonstrates:
- JavaScript interactivity
- Dynamic content
- Form validation
- SEO best practices
- Responsive design

---

## 🚀 Features Implemented

### 1. SEO Optimisation
The webpage includes:
- Proper `<title>` tag
- Meta description and keywords
- Mobile-friendly viewport settings
- Structured headings (`h1`, `h2`)
- Image alt attributes
- Internal navigation links

---

### 2. JavaScript Functionality

#### Interactive Elements
- Accordion menu for services
- Image lightbox gallery
- Button-triggered interactions

#### Dynamic Content
- Live search filter for services list
- DOM manipulation for showing/hiding content

---

### 3. Forms with Validation

#### Enquiry Form
Users can:
- Submit service enquiries
- Select a service type
- Enter messages

#### Contact Form
Users can:
- Send general messages
- Enter phone and email details
- Choose message category

#### Validation Features
- HTML5 required fields
- Email validation
- Phone number format check
- JavaScript validation for inputs

---

### 4. Responsive Design
- Mobile-friendly layout using viewport meta tag
- Flexible sections and styling
- Clean UI layout with CSS

---

### 5. Interactive UI Components
- Accordion sections for services
- Image gallery with lightbox effect
- Navigation menu for smooth section access
